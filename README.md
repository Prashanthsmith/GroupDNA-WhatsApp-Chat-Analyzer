# 📊 GroupDNA – WhatsApp Group Chat Analyzer

A Python project that analyzes exported WhatsApp chats and generates meaningful insights about user activity and communication patterns.

---

## 🚀 Features

- 📂 WhatsApp Chat Parsing
- 📈 User Activity Analysis
- 🏆 Top Active Members
- 📝 Word Frequency Analysis
- 🖼️ Media & Deleted Message Tracking
- ⏰ Hourly & Daily Activity Analysis
- 🔥 NumPy-based Activity Heatmap
- 🧠 Personality Archetype Detection
- 📊 Final Analytics Report

---

## 🛠️ Technologies Used

- Python
- NumPy
- Google Colab
- Regular Expressions (Regex)
- File Handling
- Collections (Counter & defaultdict)

---

## 📷 Sample Output

Displays:

- Most Active Member
- Least Active Member
- Hourly Activity
- Daily Activity
- Word Frequency
- Media Statistics
- Personality Detection
- Group Analytics Report
